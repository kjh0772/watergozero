# 버전 관리 노트 (관수 시스템)

## 0.2 (2024) - 시니어 친화 반응형 UI 리팩토링

- 모바일 시니어 기준 큰 글씨(20px) 및 큰 터치 영역(최소 48px) 적용
- 데스크톱(md 이상) 일반 크기(16px) 유지
- 전체 페이지 반응형 디자인 적용 (메인, 기록, 공급설정, 시스템설정)
- viewport 메타 태그 최적화 (확대 허용, 안전 영역 지원)
- 글로벌 스타일 개선 (터치 최적화, 접근성 향상)
- 모바일 카드 레이아웃 / 데스크톱 테이블 레이아웃 분리

## 0.1 (최초)

- Next.js + SQLite (시스템설정/공급설정/기록 DB)
- 메인: 현재시각, 센서 상태, 물탱크 수위, 펌프/밸브 동작상태
- 공급 기록 조회, 공급설정, 시스템설정 페이지
- 1회 공급 / 공급 정지 요청 API
- 공급 트리거 시간 슬롯 입력 개선 (추가/삭제, time 입력)
- 구역별 공급 시간 1번→14번 순차, 일괄 설정
- **실제 공급 시퀀스 구현**: 1회 공급 시 구역 1~14 순차 실행, 동작상태 실시간 반영
