# 버전 관리 노트 (관수 시스템)

## 0.3 (2026-02) - Modbus·릴레이·기록 UI 개선

- **Modbus 코일 맵핑**: 1032=펌프(P1), 1033~1047=밸브0~14(구역1~15)
- **Waveshare RPi Relay Board 8ch**: 1032~1039를 Ch1~Ch8과 동시 출력 (sysfs GPIO, udev 권한 안내)
- **시간 트리거 동작상태**: run_state DB 저장으로 메인 동작 카드에 트리거 공급 반영
- **공급 기록**: 오늘 공급 바 차트(10분 간격, 카드 100% 표시), 한국시간(UTC+9) 통일, 시작/종료 제거, 출력(Ch1~Ch16) 표시
- **문서**: MODBUS_GUIDE·DEPLOY-PI 갱신, 99-gpio-sysfs.rules

## 0.2 (2024) - 시니어 친화 반응형 UI 리팩토링

- 모바일 시니어 기준 큰 글씨(20px) 및 큰 터치 영역(최소 48px) 적용
- 데스크톱(md 이상) 일반 크기(16px) 유지
- 전체 페이지 반응형 디자인 적용 (메인, 기록, 공급설정, 시스템설정)
- viewport 메타 태그 최적화 (확대 허용, 안전 영역 지원)
- 글로벌 스타일 개선 (터치 최적화, 접근성 향상)
- 모바일 카드 레이아웃 / 데스크톱 테이블 레이아웃 분리
