# 버전 관리 노트 (관수 시스템)

## 0.1 (최초)

- Next.js + SQLite (시스템설정/공급설정/기록 DB)
- 메인: 현재시각, 센서 상태, 물탱크 수위, 펌프/밸브 동작상태
- 공급 기록 조회, 공급설정, 시스템설정 페이지
- 1회 공급 / 공급 정지 요청 API
- 공급 트리거 시간 슬롯 입력 개선 (추가/삭제, time 입력)
- 구역별 공급 시간 1번→14번 순차, 일괄 설정
- **실제 공급 시퀀스 구현**: 1회 공급 시 구역 1~14 순차 실행, 동작상태 실시간 반영
